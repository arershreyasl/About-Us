# ai4career
**Helping Engineering students land top jobs.**  
Open-source roadmaps, interview bundles, resume templates, mentorship prompts and placement AI tools.

---

## What you'll find here
- **Roadmaps:** Complete learning paths for ML/DS/Web/DSA.
- **Interview Prep:** Mock interview prompts, behavioral questions, system design notes.
- **Projects & Templates:** Real project blueprints, resume & cover letter templates.
- **Community Resources:** Events, volunteer mentors, contributor guides.
- **Open-source tools:** Small utilities, exam/quiz bots, auto-feedback scripts.
